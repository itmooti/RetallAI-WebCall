Retell Widget Embed

The simplest way to add AI conversation to any website. Just one <script> tag in your page's <head>.

// Chat Widget Snippet

<script
  id="retell-widget"
  src="https://dashboard.retellai.com/retell-widget.js"
  type="module"
  data-public-key="YOUR_RETELL_PUBLIC_KEY"
  data-agent-id="YOUR_CHAT_AGENT_ID"
  data-agent-version="0"
  data-title="Chat with us!"
></script>

// Callback Widget Snippet

<script
  id="retell-widget"
  src="https://dashboard.retellai.com/retell-widget.js"
  type="module"
  data-public-key="YOUR_RETELL_PUBLIC_KEY"
  data-agent-id="YOUR_VOICE_AGENT_ID"
  data-widget="callback"
  data-phone-number="+61XXXXXXXXX"
  data-title="Request a Call"
  data-countries="AU,US,GB"
  data-color="#6c5ce7"
></script>

// Setup Steps

1

Create a Public Key

In your Retell Dashboard, go to Keys → Add Key → select Public Key. Enter the domain where you'll embed the widget.

2

Get your Agent ID

For chat widgets, use a Chat Agent ID. For callback widgets, use a Voice Agent ID. Find these in your agent settings.

3

Replace the placeholders

Swap YOUR_RETELL_PUBLIC_KEY and YOUR_CHAT_AGENT_ID (or voice agent ID) with your actual values.

4

Drop it in

Paste the script tag into the <head> of any HTML page. A floating button will appear automatically in the bottom-right corner.

Tip: The widget uses public keys (not your API secret), so it's safe to embed directly. For optional reCAPTCHA protection, see the full widget docs.
